# rat-found-a-job

A python game where the player must search for a sewer rat in a grid(matrix) and then find his job to start earning money

# how to play 

the game is based on a grid (matrix) with hidden elements.the player must:
find the sewer rat in the grid 
then find his job 
complete both objectives within a limited number of attempts  

# goal 

find the rat. find the job. earn the money.
